# 📝 To-Do List Web Application

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.  
This project helps users manage daily tasks efficiently with a clean UI and browser-based storage.

---

## 🚀 Features

- Add new tasks easily
- Mark tasks as completed (✔) or not completed (❌)
- Delete individual tasks
- Save entire task list permanently in browser
- Data stays even after refreshing or closing the tab
- Clean and modern UI design
- Smooth hover effects and animations

---

## 🛠️ Technologies Used

- HTML5 – Structure of the application  
- CSS3 – Styling and layout  
- JavaScript (ES6) – Logic and functionality  
- LocalStorage – Store data in browser

---

## 💡 How to Use

1. Open the project in a browser  
2. Enter a task in the input field  
3. Click **Add Task** to add it to the list  
4. Use:
   - ✔ Tick button → mark as completed  
   - ❌ Cross button → mark as not completed  
   - 🗑️ Trash button → delete task  
5. Click **Save List** to store tasks permanently  
6. Click **Delete List** to clear saved data
